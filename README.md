## Описание
Приложение для сбора данных с разных датчиков и отправки их в хранилище для последующего анализа и контроля.
На данный момент моддерживается связь со следующими датчиками:
- Zont. Используется публичный API, по которому можно получить всю необходимую информацию ою установленных датчиках.
