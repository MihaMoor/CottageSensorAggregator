# CottageSensorAggregator

Приложение для сбора данных с разных датчиков и отправки их в хранилище для последующего анализа и контроля.
На данный момент моддерживается связь со следующими датчиками:
- Zont. Используется публичный API, по которому можно получить всю необходимую информацию ою установленных датчиках.

## Настройка приложения
Все настройки указываются в файле appsettings.json. Приложение поддерживает hotreload, т.е. при изменение настроек в файле appsettings.json приложение автоматически считывает их и применяет, без необходимости перезагрузки приложения.

### Kestrel
В секции Kestrel -> Http -> Url необходимо указать HOST и PORT, на котором будет запускаться приложение.

### Zont
В секции Zont необходимо заполнить поля:
- Email - почтовый адрес, по которому тех поддержка сможет с вами связаться.
- Login - логин от личного кабинета Zont.
- Password - пароль от личного кабинета Zont.
